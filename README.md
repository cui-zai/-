# 🎵 音乐推荐系统

![Python](https://img.shields.io/badge/Python-3.9-blue)
![Flask](https://img.shields.io/badge/Flask-2.3.2-green)
![License](https://img.shields.io/badge/License-MIT-yellow)
![Status](https://img.shields.io/badge/Status-Active-success)

基于云计算技术的智能音乐推荐系统，采用多种推荐算法，为用户提供个性化的音乐推荐体验。

## ✨ 功能特性

### 🎯 智能推荐引擎
- **热度推荐** - 基于整体播放量的热门歌曲
- **协同过滤** - 基于用户相似度的个性化推荐
- **内容推荐** - 基于歌曲特征相似度的推荐
- **混合推荐** - 综合多种算法的智能推荐

### 👤 用户系统
- 用户注册、登录与认证
- 个人资料管理
- 播放历史记录
- 个性化推荐偏好设置

### 🎵 音乐管理
- 歌曲库管理（增删改查）
- 多维度歌曲分类
- 实时播放统计
- 用户评分系统

### 📊 系统监控
- 实时数据统计面板
- 系统健康状态监控
- 推荐算法效果分析
- 用户行为分析

### ☁️ 云原生特性
- 基于OpenStack云平台部署
- 弹性伸缩能力
- 高可用架构设计
- 容器化部署支持
